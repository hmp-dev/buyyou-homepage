Features

경험 거래를 위한 모든 기능

인사이트 등록부터 결제, 채팅, 미팅, 정산까지 — 안전하고 편리한 경험 거래 플랫폼

인사이트 등록

당신의 전문 분야 경험, 노하우, 커리어 조언을 인사이트로 등록하고 판매하세요. 카테고리별 분류와 미팅 방식 선택이 가능합니다.

안전한 결제

KG이니시스 PG사를 통한 안전한 결제 시스템. 신용카드, 간편결제(카카오페이, 네이버페이 등)를 지원합니다.

실시간 채팅

인사이트 구매 즉시 채팅방이 자동 생성됩니다. 판매자와 실시간으로 미팅 일정과 장소를 조율하세요.

에스크로 정산

결제 금액은 미팅 완료 확인 전까지 안전하게 보관됩니다. 양측 확인 또는 24시간 자동 확정 후 판매자에게 정산됩니다.

본인인증

다날 PASS 본인인증으로 모든 회원의 실명을 확인합니다. 안전한 거래 환경을 보장합니다.

푸시 알림

인사이트 구매 알림, 새 채팅 메시지, 미팅 리마인더 등 중요한 소식을 실시간으로 받아보세요.

How it works

간단한 이용 방법

가입부터 미팅 완료까지, 5단계로 완성되는 경험 거래

1

회원가입 & 본인인증

Apple 또는 Google 소셜 로그인으로 간편 가입. 다날 PASS 본인인증으로 실명 확인 후 서비스를 이용합니다.

2

인사이트 탐색 & 구매

다양한 카테고리의 인사이트를 검색/필터링하고, 원하는 인사이트를 선착순으로 구매합니다. (KG이니시스 PG 결제)

3

채팅으로 미팅 조율

구매 즉시 자동 생성된 채팅방에서 판매자와 미팅 일정, 장소, 방식(대면/화상/전화)을 조율합니다.

4

1:1 미팅 진행

약속된 시간과 장소에서 전문가와 1:1 미팅을 진행합니다.

5

미팅 완료 & 자동 정산

양측 미팅 완료 확인 → 판매자에게 정산. 24시간 미확인 시 자동 확정됩니다.

FAQ

자주 묻는 질문

BuyYou는 어떤 서비스인가요?
전문가의 경험, 노하우, 커리어 조언 등을 "인사이트"로 등록하고, 이를 필요로 하는 사람이 구매하여 1:1 미팅(대면, 화상, 전화)을 진행하는 플랫폼입니다.
결제는 어떻게 이루어지나요?
KG이니시스 PG사를 통해 안전하게 처리됩니다. 신용카드, 카카오페이, 네이버페이 등을 지원하며, 결제 금액은 에스크로 방식으로 미팅 완료까지 안전 보관됩니다.
환불은 가능한가요?
미팅 24시간 전까지 취소 시 위약금 3,000원 차감 후 환불됩니다. 판매자 귀책(노쇼 등) 시 전액 환불됩니다.
누구나 인사이트를 판매할 수 있나요?
본인인증 완료한 만 19세 이상 회원은 누구나 등록 가능합니다. 최소 가격은 50,000원이며, 대면/화상/전화 미팅 방식을 선택할 수 있습니다.
정산은 언제 되나요?
양측 미팅 완료 확인 후 정산이 진행됩니다. 24시간 미확인 시 자동 확정. 판매자는 90%에서 PG 수수료를 차감한 금액을 받습니다.
